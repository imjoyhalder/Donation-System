What is JavaScript?
JavaScript is a scripting language used to create and control dynamic website content.
What is the difference between var, let, and const?
var is function-scoped, let is block-scoped, and const is also block-scoped but cannot be reassigned.
What is an arrow function in JavaScript?
An arrow function is a concise way to write functions using => syntax.
What is the DOM?
The DOM (Document Object Model) is a programming interface for web documents that allows scripts to update the content and structure dynamically.
What is an event listener in JavaScript?
An event listener is a function that waits for a specific event to occur on an element.
What are JavaScript promises?
A promise is an object that represents the eventual completion or failure of an asynchronous operation.
How do you handle errors in JavaScript?
Errors can be handled using try...catch blocks or by using .catch() with promises.
What is a closure in JavaScript?
A closure is a function that remembers the variables from its outer scope even after the outer function has finished executing.
What is the difference between == and === in JavaScript?
== checks for value equality with type coercion, while === checks for both value and type equality.
What is the difference between synchronous and asynchronous JavaScript?
Synchronous JavaScript executes code line by line, while asynchronous JavaScript allows non-blocking operations like API calls and setTimeout.
